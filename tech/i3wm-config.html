WIFI

Arch 系使用 NetworkManager 管理 WIFI 的连接情况。 nmcli 是自带的命令行 WIFI 管理工具。 

# 查看当前 wifi 列表
nmcli device wifi list

# 查看当前连接
nmcli connection

# 断开当前连接
nmcli connection down <WIFI SSID>

# 连接新 wifi 并且询问密码
nmcli connection up <WIFI SSID> --ask

多试试 TAB 补全，可以不用去记忆繁杂的命令。

使用 nm-applet 图形化管理 WIFI 。在 i3 的 config 文件下添加： 

exec_always nm-applet --no-agent

亮度显示

安装 xbacklight 包，配置中添加： 

bindsym XF86MonBrightnessUp exec xbacklight -inc 10 # increase screen brightness
bindsym XF86MonBrightnessDown exec xbacklight -dec 10 # decrease screen brightness

触控板 Tap to Click

参考 Enable tap to click in i3 wm 

sudo mkdir -p /etc/X11/xorg.conf.d && sudo tee <<'EOF' /etc/X11/xorg.conf.d/90-touchpad.conf 1> /dev/null
Section "InputClass"
	Identifier "touchpad"
	MatchIsTouchpad "on"
	Driver "libinput"
	Option "Tapping" "on"
EndSection

EOF
