1 基础知识

1.1 打印第一行和第二行

awk '{print $1 $2}' file

$0 表示行所有元素。

1.2 过滤记录

awk '$3 == 0 && &6 > 0 {print $0}' file

2 内建变量

内建变量 含义
$0 整个行的元素
$1~$n 当前记录的第 n 列
FS 输入字段分隔符，默认为空格或者Tab
NF 当前记录有多少字段
NR 已经读出的记录数
RS 记录分隔符，默认为换行
OFS 输出字段分隔符，默认为空格
ORS 输出记录分隔符，默认为换行

2.1 指定分隔符

awk -F: '{print $1, $3}' file
# 等价于
awk 'BEGIN{FS=":"} {print $1, $3}' file
# 多个分隔符
awk -F '[:,]'

3 字符串匹配

3.1 第六列元素不等于‘wait’或者‘fin’的行，输出其行号、第一列和第二列元素

awk '$6 !~ /wait|fin/ || NR == 1 {print NR, $1, $2}' file
